Python Programming Language

Introduction to Python

Python is a versatile, interpreted, and high-level programming language known for its simplicity and readability. Created by Guido van Rossum in 1991, Python has become one of the most popular programming languages worldwide.

Key Features:

  • General-Purpose Language
  • Readability and Simplicity
  • Interpreted Language
  • Versatile Applications
  • High-Level Language
  • Cross-Platform
  • Large Standard Library
  • Open Source
  • Dynamic Typing
  • Object-Oriented
  • Community and Ecosystem

Python's strengths lie in its ease of use, rich standard library, and active community, making it suitable for a wide range of applications, including web development, data analysis, scientific computing, machine learning, and automation.

Program 1.1 - Basic data types

  1. Integer (int):
    • Used to store whole numbers
    • age = 25
  2. Float (float):
    • Used to store decimal numbers
    • price = 10.99
  3. String (str):
    • Used to store text
    • Enclosed in a single quote ('') or double quote ("")
    • name = 'John' same as name = "John"
    • his_age = '18' (Numbers can be presented as string)
  4. Boolean (bool):
    • Used to store True or False values
    • It is important you know now that T in True is capital as well as F in False. This is a rule in python
    • Used for decision making
    • is_valid = True
  5. List (list()):
    • Used to store a collection of items, which can be different data types
    • It is enclosed in a square brackets []
    • prime_numbers = [1, 2, 3, 5, 7]
  6. Tuple (tuple()):
    • It is similar to lists but are immutable (cannot be modified once created)
    • It is enclosed in a brackets ()
    • prime_numbers = (1, 2, 3, 5, 7)
  7. Dictionary (dict()):
    • Used to store key-value pairs
    • It is enclosed in a curly brackets {}
    • person = {'name': 'Mike', "age": 20}
  8. Set (set()):
    • Used to store a collection of unique item
    • It is enclosed in a curly brackets {}
    • unique_samples = {10, 15, 63, 17, 4}
  9. NoneType (None):
    • Represents the absent of a value or a null value
    • none or NONE is unacceptable
    • result = None

Program 1.2 - Variable rules

In Python, variable names must adhere to the following rules:

  1. Variable names are case-sensitive: `myVariable` and `myvariable` are treated as different variables.
  2. Variable names can consist of letters (both uppercase and lowercase), digits, and underscores. However, a variable name cannot start with a digit.
  3. Variable names must start with a letter (a-z, A-Z) or an underscore (_). They cannot start with a digit (0-9).
  4. Variable names can contain letters, digits, and underscores. For example, `my_var_1` is a valid variable name.
  5. Python has some reserved words keywords that cannot be used as variable names because they have special meanings in the language. For example, you cannot use words like `if`, `else`, `for`, `while`, `import`, and others as variable names.
  6. Variable names should be descriptive and reflect the purpose of the variable. This helps make your code more readable and maintainable.
  7. By convention, variable names are usually written in lowercase letters with words separated by underscores (snake_case) for better readability. For example, `my_variable_name`.
  8. Although not a strict rule, Python's style guide (PEP 8) recommends using lowercase for variable names and reserving uppercase letters for constants.

Here are some examples of valid and invalid variable names:

  • my_variable
  • user_name
  • num1
  • _total
  • count123

Sample of invalid variable names:

  • $money: Invalid character
  • 2_letters: Cannot start with a number
  • user-name: Invalid character
  • import: Invalid already used as keyword

Remember that following these variable naming conventions and rules will help you write cleaner and more maintainable Python code.

1.4.3-Looping

This carries out execution in a repetitive manner. It could require going through a line of code or series of lines until the assigned task is complete. There are two types of looping:

  1. for loop: Used when you know the number of iterations in advance
  2. while loop: Used when the number of iterations depends on a condition

We can loop through a list. A list is a defined array of items

  • prime_numbers = [1, 2, 3, 5, 7, 11, 13]
  • for number in prime_numbers:
  • print(number)

This is a loop statement generated by for ... in ... statement. It is made to go through the list items one after another until the last item before it terminates. The number of items in the list is 7 and print will happen 7 times printing each item on the list one at a time.

while loop can be used to go through a block of code until a condition is satisfied. It is like a situation where someone is asked to keep doing something until a condition is met.

  • count = 0
  • while count <= 10
  • count = count + 2
  • print(count)
  • print("out of loop")

This process will always have a variable initialized so we have a starting point and a variable to compare with. Also, this variable will need to be adjusted to tend towards the endpoint (in this case 10).

Let's review the steps

  1. variable count is assigned an integer 0
  2. count which is 0 is compared to integer 10 and result is True (because 0 is less than 10)
  3. count is reassigned to 2 (0 + 2)
  4. count is sent to output
  5. The execution goes back to the while statement at step 2 and repeat to step 5 This process will continue until count becomes greater than 10. This process is called looping
