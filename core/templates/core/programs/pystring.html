Module 2: Part 1 - Python String

String will be covered in two modules as it is so broad

String are very essential and are widely used. They are stored text, characters, numbers, words, symbols and sentences, notes and even documents. It will be impossible for python or any other programming language to do much without string. Understanding string now, will help you alot to make progress in this course.

String is defined with single, double or triple quote before and after the value.

  • my_number = "25" # my_number is defined as string type due to the double quote around the value 25
  • greet = 'hello world' # Note the single quote before and after the words
  • address = """24 Terrace Avenue""" # triple quote

    • Text can have non-numeric characters including quotes, so these alternatives help alot in dealing with whatever nature of text we pose as string.

  • his_castle = "Kelvin's house" # a single quote is part of the text, so double quote is used to ensure the program does not take single quote as part of the string definition.
  • statement = 'He said, "I am going to win"' #
  • address = """24 Terrace Avenue""" # triple quote
