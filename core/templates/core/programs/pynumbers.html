Module 1: Numbers in Python

This module is designed to give learners a clear and direct understanding of the two primary numeric types in Python: integers and floating-point numbers. Our aim is to build foundational knowledge about these types, how they are defined, and how they differ from each other.

Integer: They are whole numbers without any fractional component or decimal point. They are defined in python as as follows:

  • number_of_people = 25 # Where number_of_people stored the integer 25

    • Likewise

  • population = 15245635774 # another integer defined

    • Python like to make things clear and easy for human readability and therefore the above can also be written as

  • population = 15_245_635_774 # Making it easy to know this is 15m+
  • # In python, the underscore is allowed for integers

    • It is important to know that there is no limit to the magnitude of the integer in python.

    • Floating-point: The second number type is floating-point. This is a number with decimal point or fractional component.
  • speed = 9.85 # speed variable is of the type floating-point with magnitude 9.85
  • data = 0.125 # another floating point defined
  • score = 25 # score is floating-point but of the same magnitude with number_of_people defined earlier as integer

    • We can confirm the number type with python's built-in type() function

  • my_number = 58 # This is an integer
  • # let us confirm that
  • print(type(my_number)) # -> class-int
  • area = 23.45 # This is a floating-point
  • print(type(area)) -> class_float

    • We can convert an integer to floating-point with python's built-in float() functions

  • my_number = float(my_number) # my_number is now of type flating point
  • # let us confirm it
  • print(type(my_number)) # -> class-float
  • # likewise, area which is float can become an integer
  • area = int(area) # area is now an integer
  • print(type(area)) -> class_int
  • print(area) -> 23 # which is the whole number portion of the float-point area

    • What we have covered on numbers is enough to take us to the next module which is string. String is so broad that a module is not enough. Click next to continue.

    • Key Points
    • Numbers are Integer and Floating-point. Complex number is excluded in this module
    • Integers are whole numbers while floating-point has fractional component or decimal point
    • The types can be confirmed with the type() function
    • The numbers can be inter converted with int() function from float to integer and float() function from integer to floating-point.
    • This knowledge will be greatly applied in almost all aspect of python programming
